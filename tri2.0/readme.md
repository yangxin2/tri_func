# 界面更换说明

## 原项目地址

https://github.com/yangxin2/tri_func/tree/main/Tri-Func-Calculator-CQU

## 界面更换方案

因为本组界面具有角度与弧度变换的选择，但该项目输入数据只有角度制，因此修改部分去除了界面中角度与弧度变换的选择，只能输入角度制。

在界面中依据原项目，增加了tan和arccos两个功能函数计算。

## 功能：

输入合法数据，输出计算结果

输入非法格式，输入框内显示红色字体

输入数据超出范围，输出显示Math ERROR

## 开发环境

python==3.7

sys
